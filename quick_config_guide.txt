FAHRPC Quick Config Guide
========================

This guide explains how to create a custom config.json to override default settings in FAHRPC.

How FAHRPC Configuration Works:
------------------------------
- FAHRPC uses sensible defaults from default_config.json (do not edit this file).
- If you want to change any settings, create a config.json file in your config directory.
- You only need to include the settings you want to override—missing options will use defaults.

Where is config.json?
---------------------
- Windows: %LOCALAPPDATA%\Bandokii\fahrpc\config.json
- macOS:   ~/Library/Application Support/fahrpc/config.json
- Linux:   ~/.config/fahrpc/config.json

How to Create a Custom config.json:
-----------------------------------
1. Open your config directory (see above).
2. Create a new file named config.json.
3. Add only the settings you want to change. For example:

{
  "discord": {
    "client_id": "YOUR_CUSTOM_CLIENT_ID"
  },
  "display": {
    "start_hidden": false
  }
}

- This will override the Discord client ID and set the app to show the console on startup.
- All other settings will use the defaults from default_config.json.

How to Find All Available Settings:
-----------------------------------
- See src/fahrpc/data/default_config.json for the full list of available options and their default values.
- You can copy keys from default_config.json into your config.json and change their values as needed.

Tips:
-----
- Only include keys you want to change—keep your config.json minimal.
- If you want to revert to a default value, simply remove that key from your config.json.
- If you make a mistake, delete config.json and FAHRPC will use all default settings.
